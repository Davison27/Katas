Write a program to convert Arabic numbers into their Roman numeral equivalents.

The Romans wrote their numbers using combinations of the following letters:

Number      Numeral
  1	    ->    I
  5	    ->    V
  10	  ->    X
  50	  ->    L
  100	  ->    C
  500	  ->    D
  1000  ->    M
